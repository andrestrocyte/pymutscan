# Public search benchmarks

`radius2_validation_500.tsv.gz` is a deterministic 500-sequence, 30-base DNA
dataset with abundance scores and planted distance-two relationships. It is
generated by `scripts/benchmark_algorithms.py` with seed `20260712`.

`search_benchmark.json` records median optimized/exhaustive timings, exact-map
equality, the adaptive 128/256/384 crossover, and peak traced memory. The
optimized-only 20,000-sequence profile is
included to characterize larger-scale memory without spending quadratic work
on a reference whose correctness has already been checked at three scales.

Regenerate both artifacts from the repository root:

```bash
python scripts/benchmark_algorithms.py
```
