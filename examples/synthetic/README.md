# Synthetic MAPseq example

This directory contains a small, deterministic, public-safe paired-end dataset generated by
`scripts/generate_synthetic_mapseq.py`. It exercises the default 6-nt index, 30-nt barcode,
and 16-nt UMI layout without containing experimental reads.

The positive and negative controls include exact and one-base-error indices, barcode and UMI
errors, a shared-UMI template-switch candidate, constant-region mismatches, low-quality reads,
and ambiguous bases. `truth.json` records the expected identities and QC counts.

Regenerate the files from the repository root with:

```bash
python scripts/generate_synthetic_mapseq.py
```
